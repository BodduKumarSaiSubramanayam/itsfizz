# Aetheria Full-Stack Store

A premium botanical skincare store prototype built for the Digital Heroes Shopify challenge.

## 📁 Structure
- `frontend/`: Vite + Vanilla TypeScript project for the UI.
- `backend/`: Node.js + Express API for product data and services.

## 🚀 How to Run
1. **Initial Setup**:
   ```bash
   npm run install-all
   ```
2. **Start Both (Frontend + Backend)**:
   ```bash
   npm start
   ```

## ✨ Features Implemented
- **Premium Design System**: Using the 60/30/10 color rule.
- **Dynamic Product Loading**: Fetched from the custom Express API.
- **Sticky Add-to-Cart**: High-conversion UX element.
- **Trust Badges**: Integrated SVG badges for brand authority.
- **Newsletter Signup**: Functional endpoint for capturing leads.
- **Responsive Layout**: Optimized for mobile and desktop.
